# Two Sum

## Description

Given an integer `y`, return true if y is a palindrome.

## Test Cases

### Test Case 1
*Input:*
```json
{
  "y": 10
}
```
*Output:*
```json
[false]
```
### Test Case 2
*Input:*
```json
{
  "y": 101
}
```
*Output:*
```json
[true]
```
### Test Case 3
*Input:*
```json
{
  "y": -121
}
```
*Output:*
```json
[false]
```
### Test Case 4
*Input:*
```json
{
  "y": 324423
}
```
*Output:*
```json
[true]
```
