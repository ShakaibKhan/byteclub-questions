# Two Sum

## Description

Given two sorted arrays <code>arr1</code> and <code>arr2</code> return both arrays merged.

## Test Cases

### Test Case 1
*Input:*
```json
{
  "arr1": [2, 7, 11, 15],
  "arr2": [3, 8, 14]
}
```
*Output:*
```json
[2, 3, 7, 8, 11, 14, 15]
```

### Test Case 2
*Input:*
```json
{
  "arr1": [1, 3, 10, 15],
  "arr2": [16, 81, 91]
}
```
*Output:*
```json
[1, 3, 10, 15, 16, 81, 91]
```
### Test Case 3
*Input:*
```json
{
  "arr1": [1, 5, 8, 10, 100],
  "arr2": [4, 6, 11, 12]
}
```
*Output:*
```json
[1, 4, 5, 8, 10, 11, 12, 100]
```
